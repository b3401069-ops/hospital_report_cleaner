================================================
  醫院報表清洗工具 使用說明
================================================

【功能說明】
  將多個 Excel 報表合併成一個統一格式的報表
  - 自動合併重複欄位
  - 保留所有不重複欄位
  - 支援 .xls / .xlsx / .csv 格式

【系統需求】
  - Windows 7 以上 / macOS / Linux
  - Python 3.8 以上（免費下載：https://www.python.org/downloads/）
  - 安裝 Python 時請勾選「Add Python to PATH」

【使用步驟】

  第一步：放入報表
  ─────────────────
  將您的 Excel 報表檔案複製到「raw」資料夾中
  
  例如：
  raw/
  ├── 報表A.xls
  ├── 報表B.xlsx
  └── 報表C.csv

  第二步：執行程式
  ─────────────────
  Windows：雙擊「執行報表清洗.bat」
  Mac/Linux：在終端機執行 python3 main.py

  第三步：查看結果
  ─────────────────
  結果檔案在 output/cleaned_reports.xlsx
  包含以下工作表：
  - cleaned_data：所有欄位的大聯集（您要的！）
  - standardized_data：標準化後的資料
  - patient_master：病患主檔
  - patient_list：病患清單
  - summary_zh：統計摘要

【資料隱私提醒】
  ⚠️  強烈建議先將報表中的敏感資訊匿名化：
  - 病歷號 → 替換為隨機編號
  - 姓名 → 替換為假名
  - 身分證號 → 替換為隨機編號
  - 電話/地址 → 替換為假資料

【新增報表】
  有新報表要處理時：
  1. 將新報表放入「raw」資料夾
  2. 重新執行程式
  3. 結果會自動包含新報表的所有欄位

【常見問題】

  Q: 執行時出現「python 不是內部或外部指令」
  A: Python 未加入系統路徑，請重新安裝並勾選「Add Python to PATH」

  Q: 出現「ModuleNotFoundError」
  A: 請確認已執行「執行報表清洗.bat」（會自動安裝套件）

  Q: 報表欄位沒有被正確合併
  A: 可在 mapping/column_mapping.csv 中新增欄位對照規則

  Q: 記憶體不足
  A: 減少一次處理的報表數量，或分批處理

【檔案結構說明】
  dist_package/
  ├── main.py                 ← 主程式
  ├── requirements.txt        ← Python 套件清單
  ├── 執行報表清洗.bat        ← Windows 一鍵執行
  ├── src/                    ← 程式原始碼
  │   └── report_cleaner/
  ├── mapping/                ← 欄位對照表
  │   ├── column_mapping.csv  ← 欄位名稱對照
  │   └── value_mapping.csv   ← 欄位值對照
  ├── raw/                    ← 放入報表檔案
  └── output/                 ← 輸出結果
